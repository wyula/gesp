### 【题目描述】

今天是星期$X$，小杨还有$N$天就要考试了，你能推算出小杨考试那天是星期几吗？（本题中使用$7$表示星期日）

### 【输入】

输入$2$行，第一行一个整数$X$（$1 ≤ X ≤ 7$）；第二行一个整数$N$（$1 ≤ N ≤ 100$）。

### 【输出】

输出一个整数，表示小杨考试那天是星期几。

### 【输入样例 1】
```
1
6
```

### 【输出样例 1】
```
7
```

### 【输入样例 2】
```
5
3
```

### 【输出样例 2】
```
1
```

### 【样例解释】

对于样例$1$：今天是星期一（$1$），过$6$天是星期日（$7$）。

对于样例$2$：今天是星期五（$5$），过$3$天是星期一（$1$）。

### 【数据范围】

`1 ≤ X ≤ 7, 1 ≤ N ≤ 100。`
