### 【题目描述】

小A最近刚刚学习了因数的概念，具体来说，如果一个正整数$a$可以被另一个正整数$i$整除，那么我们就说$i$是$a$的因数。

请你帮忙写一个程序，从小到大输出正整数$a$的所有因数。

### 【输入】

输入一行一个正整数$a$。保证$1 ≤ a ≤ 1000$。

### 【输出】

输出若干行，为$a$的所有约数，从小到大排列。

### 【输入样例 1】
```
1
```

### 【输出样例 1】
```
1
```

### 【输入样例 2】
```
6
```

### 【输出样例 2】
```
1
2
3
6
```

### 【输入样例 3】
```
10
```

### 【输出样例 3】
```
1
2
5
10
```

### 【数据范围】

`1 ≤ a ≤ 1000。`
