### 【题目描述】

小杨有一个正整数$n$，他想知道$n$是否是一个立方数。

一个正整数$n$是立方数当且仅当存在一个正整数$i$满足$i^3 = n$。

### 【输入】

第一行包含一个正整数$n$。

### 【输出】

如果正整数$n$是一个立方数，输出`Yes`，否则输出`No`。

### 【输入样例 1】
```
8
```

### 【输出样例 1】
```
Yes
```

### 【输入样例 2】
```
9
```

### 【输出样例 2】
```
No
```

### 【样例解释】

对于样例1，存在正整数$i=2$使得$2^3=8$，因此$8$为立方数。

对于样例2，不存在满足条件的正整数，因此$9$不为立方数。

### 【数据范围】

`1 ≤ n ≤ 1000。`
