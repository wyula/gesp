（样题）提出“存储程序”的计算机工作原理的是( )。
A 克劳德·香农
B 戈登·摩尔
C 查尔斯·巴比奇
D 冯·诺依曼
【解析】冯·诺依曼提出了"存储程序"的计算机工作原理，即程序和数据都以二进制形式存储在存储器中，计算机按顺序执行指令。
课堂练习 存储器是计算机系统中的记忆设备，它主要用来( )。
A 存放数据
B 存放程序
C 存放数据和程序
D 存放微程序
【解析】存储器的主要功能是保存各类程序和数据信息，包括数据和程序代码。
课堂练习 断电后会丢失数据的存储器是( ) 。
A RAM
B ROM
C 硬盘
D 光盘
【解析】RAM（随机存取存储器）是易失性存储器，断电后数据会丢失。ROM、硬盘、光盘都是非易失性存储器。
课堂练习 RAM 既可以读取数据，又可以写入数据，ROM只能读取数据，不能写入数据。
A 正确
B 错误
【解析】RAM（随机存取存储器）可以读写数据，断电后数据丢失。ROM（只读存储器）只能读取数据，一般不能写入，断电后数据不丢失。
（2023年3月）以下存储器中的数据不会受到附近强磁场干扰的是( )。
A 硬盘
B U盘
C 内存
D 光盘
【解析】光盘使用激光读取数据，不受磁场影响。硬盘、U盘、内存都使用磁存储或半导体存储，可能受到磁场干扰。
课堂练习 计算机存储数据的基本单位是( )
A bit
B Byte
C GB
D KB
【解析】计算机存储数据的基本单位是字节(Byte)，一个字节由8个二进制位(bit)组成。
课堂练习 8个字节含( )个二进制位。
A 8
B 16
C 32
D 64
【解析】1字节(Byte)=8位(bit)，所以8字节=8×8=64位。
（2023年3月）明明和笑笑在"小庙会"上分别抽到一个4GB和4096MB的U盘，容量大的盘是笑笑的。
A 正确
B 错误
【解析】1GB=1024MB，所以4GB=4×1024=4096MB。两个U盘容量相同，并不是笑笑的更大。
（2023年6月）诞生于1986年的中华学习机CEC-I入选了2021年的CCF计算机历史记忆（一类），它的内存只有64KB。当时的汉字编码字符集GB2312中共有6763个汉字，假如每个汉字用2个字节编码，将整个GB2312汉字字符集都放入CEC-I的内存，也只占用了不超过1/5的内存空间。
A 正确
B 错误
【解析】6763个汉字，每个2字节，共6763×2=13526字节≈13.2KB。64KB的1/5是12.8KB。13.2KB>12.8KB，所以占用超过了1/5的内存空间，说法错误。
（2023年3月）下列关于C++语言的叙述，不正确的是( )。
A double类型的变量占用内存的大小是浮动的
B bool类型的变量占用1字节内存
C int类型的变量取值范围不是无限的
D char类型的变量有256种取值
【解析】double类型在C++中占用固定的8字节内存，不是浮动的。bool占用1字节，int取值范围有限，char有256种取值（0-255）。
（2023年9月）神威·太湖之光超级计算机是中国自主研制的超级计算机，在全球超级计算机TOP500排行榜中多次荣膺榜首。
A 正确
B 错误
【解析】神威·太湖之光是中国自主研制的超级计算机，使用国产处理器，多次在全球超级计算机TOP500排行榜中位列第一。

A 集成电路
