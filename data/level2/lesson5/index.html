（2023年12月）在C++代码中，执行 srand(0) 后连续两次执行 rand() 的结果相等。
A 正确
B 错误
【解析】srand(0)设置固定种子后，每次程序运行时rand()生成的随机数序列相同，但连续两次调用rand()的结果是不同的，因为随机数生成器会内部状态。
课堂练习 如下选项中，哪个能随机生成4~25之间的一个整数( )。
A 4 + rand( ) % 25
B 4 + rand( ) % (25-4)
C 4 + rand( ) % 22
D rand( ) % (25-4+1)
【解析】生成a~b范围内随机整数的公式是：a + rand() % (b-a+1)。对于4~25，范围是25-4+1=22，所以公式是4 + rand() % 22。
