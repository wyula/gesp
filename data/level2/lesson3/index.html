课堂练习 计算机网络最核心的功能是( )。
A 预防病毒
B 数据通信和资源共享
C 信息浏览
D 下载文件
【解析】计算机网络最核心的功能是数据通信和资源共享，通过网络可以实现信息传递和软硬件资源共享。
（2023年9月）我们常说的互联网（Internet）是一个覆盖全球的广域网络，它不属于任何一个国家。
A 正确
B 错误
【解析】互联网是覆盖全球的广域网络，它不属于任何一个国家所有，是由世界各国共同建设和管理的。
课堂练习 局域网的覆盖范围是( )。
A 一栋楼
B 一条街
C 一座城市
D 一个国家
【解析】局域网（LAN）覆盖范围通常是几百米或者几千米，作用范围可以是一个学校、一栋楼或者一个工厂等。
课堂练习 广域网的英文缩写是( )。
A LAN
B WAN
C MAN
D LNA
【解析】广域网的英文是Wide Area Network，缩写为WAN。LAN是局域网，MAN是城域网。
课堂练习 局域网和广域网的主要区别是( )。
A 覆盖范围
B 传输速度
C 传输介质
D 网络协议
【解析】局域网和广域网的主要区别是覆盖范围。局域网覆盖范围较小（几百米到几千米），广域网覆盖范围较大（几千公里甚至几万公里）。
课堂练习 蓝牙和Wi-Fi都是( )设备。
A 无线广域网
B 无线城域网
C 无线局域网
D 无线路由器
【解析】蓝牙和Wi-Fi都是无线局域网技术，覆盖范围较小，适合短距离通信。
课堂练习 TCP/IP四层模型中，包括网络接口层、网络层、传输层和( )。
A 物理层
B 表示层
C 会话层
D 应用层
【解析】TCP/IP四层模型从下到上依次是：网络接口层、网络层、传输层、应用层。物理层、表示层、会话层是OSI七层模型中的层次。
（2023年9月）TCP/IP的传输层的两个不同的协议分别是UDP和TCP。
A 正确
B 错误
【解析】TCP/IP传输层的两个主要协议是TCP（传输控制协议，面向连接，可靠传输）和UDP（用户数据报协议，无连接，不可靠传输）。
（样题）万维网WWW中存储了海量的数据资源，这里用于传输控制的协议是( )。
A URL
B SMTP
C HTTP
D HTML
【解析】万维网使用HTTP（超文本传输协议）进行数据传输。URL是统一资源定位符，SMTP是邮件传输协议，HTML是超文本标记语言。
课堂练习 TCP/IP四层模型中，网络接口层负责提供数据链路层和物理层的实现。
A 正确
B 错误
【解析】TCP/IP四层模型中的网络接口层对应OSI七层模型中的物理层和数据链路层，负责硬件接口和数据帧的传输。
课堂练习 TCP/IP模型中，传输层的主要协议是什么( )。
A HTTP
B FTP
C TCP和UDP
D DNS
【解析】TCP/IP模型传输层的主要协议是TCP（传输控制协议）和UDP（用户数据报协议）。HTTP、FTP、DNS都是应用层协议。
