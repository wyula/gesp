（样题）汉字没有对应的ASCII码。
A 正确
B 错误
【解析】ASCII码只包含0~127的字符，包括英文字母、数字和基本符号，不包含汉字。汉字使用GB2312、GBK、UTF-8等编码。
（2023年6月）如果 a 为 char 类型的变量，下列哪个表达式可以正确判断"a 是数字"?( )
A '0' <= a && a <= '9'
B '1' <= a && a <= '0'
C '0' <= a <= '9'
D '1' <= a <= '0'
【解析】判断字符是否为数字，需要判断其ASCII码是否在'0'(48)到'9'(57)之间。选项A使用逻辑与运算符&&连接两个比较条件，是正确的写法。选项C的写法在C++中会先计算'0'<=a，结果为bool类型，再与'9'比较，是错误的。

（2023年3月）如果a为char类型的变量，且取值为小写字母，则执行语句a=a-'a'+'A';后，a的值会变为与原值对应的大写字母。

（2023年3月）表达式(10.0 / 2)的计算结果为5.0，且结果类型为double。
A 正确
B 错误
【解析】10.0是double类型，2是int类型。在运算时，int会自动转换为double，所以结果是5.0，类型为double。
（2023年3月）如果a、b、c和d都是int类型的变量，则下列哪个表达式能够正确计算他们的平均值?( )
A (a+b+c+d)/4
B (a+b+c+d)%4

（2023年6月）下列关于C++语言的叙述，正确的是?( )

（2023年6月）如果用两个int类型的变量a和b分别表达平行四边形的两条边长，用int类型的变量h表达a边对应的高，则下列哪个表达式不能用来计算b边对应的高?( )
A a/b*(0.0+h)
B (0.0+a*h)/b
C a*h/(b+0.0)
D (a+0.0)*h/b
【解析】平行四边形面积S=a*h=b*h_b，所以b边的高h_b=a*h/b。选项A先计算a/b（整数除法会截断），再乘以(0.0+h)，结果错误。其他选项都能正确计算。

（2023年6月）如果a为int类型的变量，b为char类型的变量，则下列哪个语句不符合C++语法( )。
A a=a+1.0;
B a=(int)(b-'0');
C b=(char)(a+'0')。
D (int)b=a;
【解析】选项D错误。强制类型转换的结果是右值，不能作为赋值语句的左值。正确的写法应该是b=(char)a;或者a=(int)b;。
（2023年6月）在下列代码的横线处填写( )，使得输出是9。 #include using namespace std; int main(){ char a='3', b='6'; cout<<________; return 0; }
A (a+b)
B (a+b-'0')
C (char)(a+b)
D (char)(a+b-'0')
【解析】'3'的ASCII码是51，'6'是54。a+b=105。要输出字符'9'（ASCII码57），需要计算：a+b-'0'=51+54-48=57='9'。再强制转换为char类型即可输出字符9。
