课堂练习 计算机硬件能直接识别并执行的语言是( )。
A 高级语言
B 汇编语言
C 机器语言
D 符号语言
【解析】机器语言是计算机硬件能直接识别和执行的语言，它由二进制指令组成。高级语言和汇编语言都需要编译或汇编后才能执行。
（2023年9月）C++是一种高级程序设计语言。
A 正确
B 错误
【解析】C++是一种面向对象的高级程序设计语言，它提供了封装、继承、多态等面向对象特性。
（2023年6月）高级语言编写的程序需要经过以下( )操作，可以生成在计算机上运行的可执行代码。
A 编辑
B 保存
C 调试
D 编译
【解析】高级语言编写的程序需要经过编译，将源代码转换为机器语言，生成可执行文件后才能在计算机上运行。
（2023年3月）下列流程图，属于计算机的哪种程序结构( )。
A 顺序结构
B 循环结构
C 分支结构
D 数据结构
【解析】流程图中有条件判断和两条分支路径，属于分支结构（选择结构）。分支结构根据条件判断结果选择不同的执行路径。
（2023年6月）能够实现下面流程图功能的伪代码是( )。
A if 条件判断 then 语句块
B if 条件判断 then 什么也不做 else 语句块
C while 条件判断 do 语句块
D while not 条件判断 do 语句块
【解析】流程图中条件不成立时执行语句块，条件成立时跳过，即"while not 条件判断 do 语句块"。
（2023年9月）在特殊情况下流程图中可以出现三角框和圆形框。
A 正确
B 错误
【解析】流程图通常使用矩形框（处理）、菱形框（判断）、圆角矩形框（开始/结束）等标准符号。三角框和圆形框不是标准流程图符号。
（2023年9月）如果要找出整数a、b中较大一个，通常要用下面哪种程序结构？( )
A 顺序结构
B 循环结构
C 分支结构
D 跳转结构
【解析】比较两个数的大小需要进行条件判断，根据判断结果选择输出较大的数，这需要使用分支结构（if-else语句）。
（2023年3月）在C++语言中，标识符中可以有下划线_，但不能以下划线开头。
A 正确
B 错误
【解析】C++中标识符可以以下划线开头，但以下划线开头且后跟大写字母或另一个下划线的标识符通常是保留的。一般情况下，标识符可以以下划线开头。
（2023年3月）如果a是int类型的变量，而且值为1，则表达式'a'的值为'1'。
A 正确
B 错误
【解析】表达式'a'是一个字符常量，值为字符'a'的ASCII码（97），与变量a的值无关。字符'1'的ASCII码是49。
（2023年3月）C++语言中，>=是运算符，但=>不是。
A 正确
B 错误
【解析】>=是C++中的大于等于比较运算符，而=>不是C++的运算符。在某些语言（如C#）中=>用作Lambda表达式，但C++使用->或其他语法。
（2023年3月）在C++语言中，一个程序不能有多个main函数。
A 正确
B 错误
【解析】main函数是C++程序的入口点，一个程序只能有一个main函数。如果有多个main函数，编译器会报错。
（2023年3月）以下哪个不是C++语言的运算符( )。
A \=
B /=
C -=
D !=
【解析】/=是除等运算符，-=是减等运算符，!=是不等于运算符。\=不是C++的运算符，反斜杠在C++中用于转义字符。
（2023年3月）以下哪个不是C++语言的关键字( )。
A return
B max
C else
D case
【解析】return、else、case都是C++关键字。max不是关键字，而是标准库中的函数模板。
（2023年3月）如果a和b都是char类型的变量，下列哪个语句不符合C++语法？( )
A b = a + 1；
B b = a + '1'；
C b = 'a'++；
D b = a++;
【解析】'a'是一个字符常量，常量不能被递增（++），因为++运算符需要修改操作数的值。选项C中对常量'\'a'使用++是非法的。
（2023年3月）如果a为int类型的变量，且a的值为9，则执行a-=3；之后，a的值会是( )。
A 3
B 6
C 9
D 12
【解析】a-=3等价于a = a - 3。a初始值为9，执行后a = 9 - 3 = 6。
（2023年3月）如果a和b均为int类型的变量，下列表达式能正确判断"a等于0或b等于0"的是( )。
A (!a) || (!b)
B (a==b==0)
C (a==0) && (b==0)
D (a==0) - (b==0) == 0
【解析】!a表示a为0时为true，!b表示b为0时为true。(!a) || (!b)表示a等于0或b等于0。选项B中a==b==0会先计算a==b，结果与0比较，不符合逻辑。选项C是"a等于0且b等于0"。
（2023年6月）++和==都是C语言的运算符，但+=不是。
A 正确
B 错误
【解析】+=是C语言（也是C++）的复合赋值运算符，表示加法赋值。++是自增运算符，==是相等比较运算符。
（2023年6月）以下哪个不是C++语言的运算符？( )
A >=
B /=
C ||
D <>
【解析】>=是大于等于运算符，/=是除等运算符，||是逻辑或运算符。<>不是C++的运算符，在Pascal等语言中表示不等于，但C++使用!=表示不等于。
（2023年6月）如果a和b均为int类型的变量，下列表达式能正确判断"a等于1且b等于1"的是( )。
A (a==b) && (b==1)
B (a && b)
C (a == b == 1)
D (a * b == 1)
【解析】(a==b) && (b==1)表示a等于b且b等于1，即a等于1且b等于1。选项B只判断a和b非零。选项C中a==b==1会先计算a==b，结果与1比较。选项D中a*b==1可能有其他解（如a=1,b=1或a=-1,b=-1）。
（2023年9月）C++表达式(2 * 3) || (2 + 5)的值为67。
A 正确
B 错误
【解析】||是逻辑或运算符，结果是布尔值（true或false）。(2*3)=6，非零值为true。由于短路求值，||左边为true时不再计算右边，整个表达式的值为true（1）。
（2023年9月）如果a为int类型的变量，则表达式(a >= 5 && a <= 10)与(5 <= a <= 10)的值总是相同的。
A 正确
B 错误
【解析】表达式5 <= a <= 10会先计算5 <= a，结果是0或1，再与10比较。由于0和1都小于等于10，所以这个表达式的值永远为true。而(a >= 5 && a <= 10)只有当a在5到10之间时才为true。
（2023年9月）以下不是C++关键字的是( )。
A continue
B cout
C break
D goto
【解析】continue、break、goto都是C++关键字。cout是标准库中的对象（std::cout），不是关键字。
