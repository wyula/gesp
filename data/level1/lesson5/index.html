（2023年3月）表达式((3 == 0) + 'A' + 1 + 3.0)的结果类型为（ ）
A double
B int
C char
D bool
【解析】本题考查C++的类型转换规则。表达式中包含double类型（3.0），根据类型转换规则，所有操作数都会转换为double类型，所以结果类型为double。
（2023年6月）如果a、b和c都是int类型的变量，下列哪个语句不符合C++语法？（ ）
A a=(b==c);
B b=5.5;
C c=a+b+c;
D a+c=b+c;
【解析】本题考查C++赋值语句的语法。赋值运算符的左边必须是左值（可以被赋值的变量），选项D中a+c是表达式，不是左值，所以不符合语法。
（样题）如果a为int类型的变量，下列表达式不能正确表达"a是奇数时结果为0，否则结果非0"的是（ ）
A a%=2
B a/2*2==a
C a%2==0
D (a+1)%2
【解析】本题考查判断奇数和偶数的表达式。选项A：a%=2是赋值语句，不是表达式，且当a为奇数时a%=2结果为1（非0），当a为偶数时结果为0，与题目要求相反。
（2023年6月）如果a和b为int类型的变量，则表达式a=b可以判断a和b是否相等。
A 正确
B 错误
【解析】本题考查赋值运算符和比较运算符的区别。a=b是赋值语句，将b的值赋给a；判断相等需要使用==运算符，即a==b。
（2023年6月）如果a为int类型的变量，则表达式(a%4==2)可以判断a的值是否为偶数。
A 正确
B 错误
【解析】本题考查偶数的判断方法。表达式(a%4==2)只能判断a除以4余2，例如6%4==2，但4%4==0，所以不能判断所有偶数。判断偶数应该使用a%2==0。
（样题）如果a为int类型的变量，且表达式(a%4==0)的计算结果为真，说明a的值是4的倍数。
A 正确
B 错误
【解析】本题考查取余运算的含义。a%4==0表示a除以4余数为0，说明a是4的倍数。
（2023年3月）如果a和b均为int类型的变量，下列表达式不能正确判断"a等于0且b等于0"的是（ ）
A (a == 0) && (b == 0)
B (a == b == 0)
C (!a) && (!b)
D (a == 0) + (b == 0) == 2
【解析】本题考查逻辑表达式的运算顺序。选项B中(a == b == 0)的运算顺序是先计算a==b，得到bool值0或1，再将结果与0比较，不能正确判断a和b都等于0。
（2023年6月）如果a和b均为int类型的变量，下列表达式能正确判断"a等于0且b等于0"的是（ ）
A (a == b == 0)
B !(a || b)
C (a + b == 0)
D (a == 0) + (b == 0)
【解析】本题考查逻辑表达式。选项B：!(a || b)表示a和b都为假（即都为0），可以正确判断a等于0且b等于0。
（2023年9月）C++表达式 2-1&&2%10 的值是（ ）
A 0
B 1
C 2
D 3
【解析】本题考查运算符优先级和逻辑运算。算术运算符优先级高于逻辑运算符，所以先计算2-1=1和2%10=2，然后1&&2=1（非0值在逻辑运算中视为真）。
（2023年9月）在C++语言中，int类型的变量x、y、z的值分别为2、4、6，以下表达式的值为真的是（ ）
A x>y||x>z
B x!=z-y
C z>y+x
D x
【解析】本题考查逻辑表达式的计算。选项D：x
（2023年3月）如果a为int类型的变量，则表达式(a / 4 == 2)和表达式(a >= 8 && a <= 11)的结果总是相同的。
A 正确
B 错误
【解析】本题考查整数除法。a/4==2表示a除以4的商为2，即a在[8, 12)范围内（8<=a<12），而a>=8 && a<=11表示a在[8, 11]范围内。当a=11时，两个表达式都为真；当a=12时，a/4==2为假（12/4=3），a>=8 && a<=11也为假。所以两个表达式结果总是相同。
