（2023年9月）下面C++代码执行后的输出是（ ）。 int n=5; int cnt=1; while(n>=0){ cnt+=1; n-=2; } cout<
A 3
B 4
C 6
D 7
【解析】初始cnt=1, n=5。循环执行过程：n=5→cnt=2,n=3；n=3→cnt=3,n=1；n=1→cnt=4,n=-1。n=-1时条件不成立，循环结束，cnt=4。
（样题）能用while语句编写的循环，就可以使用for语句编写出具有同样功能的循环。
A 正确
B 错误
【解析】for循环和while循环都可以有循环变量、循环条件、循环步长，两种循环可以互相转换，同一种功能两种循环都能实现。
（2023年6月）do…while语句的循环体至少会执行一次。
A 正确
B 错误
【解析】do-while循环先执行一次循环体，再判断条件是否成立。无论条件是否满足，循环体至少会执行一次。
（2023年9月）在C++语言中，do-while循环不可能导致死循环，但while有可能。
A 正确
B 错误
【解析】do-while循环同样可能导致死循环。只要循环条件永远为true，无论是while还是do-while都会陷入死循环。
