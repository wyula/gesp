知识讲解

1、科学计算

n 科学计算又称数值计算，它是计算机最早的应用领域。

n 科学计算是指计算机用于完成科学研究和工程技术中所提出的数学

问题的计算。

n 这类计算往往公式复杂，难度很大，一般计算工具或人力难以完成。

n 例如：导弹试验、卫星发射、灾情预测；数学、物理、化学、天文

等学科的科学研究。

知识讲解

2、数据处理

n 数据处理是目前计算机最广泛的应用领域，用于信息的收集、分类、

整理、加工、存储等。

n 处理的原始数据量大，运算简单，具有大量的逻辑判断运算。

n 例如：人口统计、办公自动化、企业管理、邮政业务、机票订购、

情报检索、图书管理等。

知识讲解

3、过程控制

n 过程控制又称实时控制，用计算机实时控制对象。

n 及时收集并检测数据，按最佳值调节控制对象。

n 例如：电力、机械制造、化工、冶金、交通、军事上的导弹控制等。

知识讲解

4、计算机辅助技术

n 计算机辅助设计、制造、教学、出版、管理等

n 用模型模拟现实，用计算、逻辑判断功能模拟人脑。

n 例如：建筑工程设计、服装设计、机械制造设计、船舶设计、教育教

学、出版印刷等。

知识讲解

5、人工智能

n 人工智能（Artificial Intelligence, AI）是指让计算机模拟人类的某些

智能活动。

n 例如，可以用计算机模拟人脑的部分功能进行思维、学习、推理、联

想和决策，使计算机具有一定的“思维能力”。

n 用于各种专家系统和机器人构造等。
