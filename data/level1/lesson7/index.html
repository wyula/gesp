（2023年9月）下面C++代码执行后的输出是（ ） int m=14; int n=12; if(m%2&&n%2) cout<<"都是偶数"; else if(m%2==1&&n%2==1) cout<<"都是奇数"; else cout<<"不都是偶数或奇数";
A 都是偶数
B 都是奇数
C 不都是偶数或奇数
D 以上说法都不正确
【解析】本题考查if-else多分支语句和逻辑运算。m=14，m%2=0；n=12，n%2=0。所以m%2&&n%2=0&&0=0（假），m%2==1&&n%2==1=0&&0=0（假），执行else，输出"不都是偶数或奇数"。

A 都是偶数
B 都是奇数
C 不都是偶数或奇数
D 以上说法都不正确
【解析】本题考查if-else多分支语句。m=14，n=12，都是偶数，所以m%2==1&&n%2==1为假，两个条件都不满足，执行else，输出"不都是偶数或奇数"。注意：代码中有逻辑错误，第一个条件判断奇数却输出"都是偶数"。
（2023年9月）下面C++代码执行后的输出是（ ） int m=7; if(m/5||m/3) cout<<0; else if(m/3) cout<<1; else if(m/5) cout<<2; else cout<<3;
A 0
B 1
C 2
D 3
【解析】本题考查if-else多分支语句和整数除法。m=7，m/5=1（整数除法），m/3=2。由于||的短路求值，m/5||m/3=1||...=1（真），所以输出0。
