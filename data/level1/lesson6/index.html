课堂练习 已知已定义整型变量w,x=0,y=0,z=0, 执行表达式 w=++x||y++&&++z 后，w的值为( ) 。
A 0
B 1
C 2
D 3
【解析】本题考查逻辑运算符的短路求值。++x是前缀自增，x变为1，表达式值为1（真）。由于||的短路求值，后面的y++&&++z不会执行，所以w=1。
课堂练习 已知整型变量a=5,b=3,表达式 a+=a*=++b*3的值为( ) 。
A 65
B 80
C 100
D 120
【解析】本题考查运算符优先级和复合赋值。++b*3先执行，b变为4，4*3=12。然后a*=12，a=5*12=60。最后a+=60，a=60+60=120。
（2023年9月）C++语句cout<<(2, 3, "23")的输出为2，3，23。
A 正确
B 错误
【解析】本题考查逗号运算符。逗号运算符的结果是最后一个表达式的值，所以(2, 3, "23")的结果是字符串"23"，输出为"23"，而不是"2，3，23"。
（2023年9月）对int类型的变量a、b、c，下列语句不符合C++语法是（ ）
A c+=5;
B b=c%2.5;
C a=(b=3, c=4, b+c);
D a-=a=(b=6)/(c=2);
【解析】本题考查取余运算的操作数类型。取余运算符%要求两个操作数都必须是整型，选项B中2.5是double类型，所以不符合语法。
（2023年3月）if语句中的条件表达式的结果必须为bool类型。
A 正确
B 错误
【解析】本题考查if语句的条件表达式。if语句的条件表达式可以是任意类型，C++会将非零值视为真，零值视为假。
（2023年6月）if语句可以没有else子句。
A 正确
B 错误
【解析】本题考查if语句的语法。if语句可以单独使用，不需要else子句，当条件成立时执行if后的语句，条件不成立时跳过。
课堂练习 以下代码的输出结果是什么( ) int i=1,j=2,k=3; i==1?(i=j+k,j=5):(k++,k++); cout<
A 1 2 4
B 1 2 5
C 5 5 3
D 5 5 5
【解析】本题考查条件运算符。条件i==1为真，执行(i=j+k,j=5)。逗号表达式先执行i=j+k，i=2+3=5；然后执行j=5。由于条件为真，不执行(k++,k++)，k保持为3。输出为"5 5 3"。
