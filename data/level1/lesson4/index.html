（2023年6月）表达式(4*(11+12)/4)的计算结果为（ ）
A 47
B 20
C 23
D 56
【解析】本题考查表达式计算。先计算括号内：11+12=23，然后4*23=92，最后92/4=23。
（2023年3月）如果用两个int类型的变量a和b分别表达长方形的长和宽，则下列哪个表达式不能用来计算长方形的周长？（ ）
A a+b*2
B 2*a+2*b
C a+b+a+b
D b+a*2+b
【解析】本题考查运算符优先级和周长计算公式。选项A中，由于*的优先级高于+，所以a+b*2等价于a+(b*2)，只计算了一条长和两条宽，不能正确计算周长。
（2023年6月）如果用一个int类型的变量a表示正方形的边长，则下列哪个表达式不能用来计算正方形的面积？（ ）
A a*a
B 1*a*a
C a^2
D a*2*a/2
【解析】本题考查运算符的含义。在C++中，^是位异或运算符，不是乘方运算符。选项C中的a^2不能计算正方形的面积，应该使用a*a。
（2023年3月）如果a为int类型的变量，下列哪个表达式可以正确求出满足"大于等于a且是4的倍数"的整数中最小的？（ ）
A a*4
B a/4*4
C (a+3)/4*4
D a-a%4+4
【解析】本题考查整数除法和取余运算。选项C：(a+3)/4*4可以正确求出大于等于a且是4的倍数的最小整数。当a=3时，(3+3)/4*4=6/4*4=1*4=4；当a=4时，(4+3)/4*4=7/4*4=1*4=4。
（2023年3月）如果a为int类型的变量，则赋值语句 a=a+3;是错误的，因为这条语句会导致a无意义。
A 正确
B 错误
【解析】本题考查赋值语句的含义。a=a+3;是合法的赋值语句，它将变量a原来的值加3后再赋给a，使a的值增加3。
（2023年3月）如果a、b和c都是int类型的变量，下列哪个语句不符合C++语法？（ ）
A c=a+b;
B c+=a+b;
C c=a=b;
D c=a++b;
【解析】本题考查C++语法。选项D中的a++b是非法的，++是单目运算符，不能放在两个变量之间。正确的写法应该是a++或++a。
（2023年3月）如果a为int类型的变量，且a的值为6，则执行a*=3;之后，a的值会是（ ）
A 3
B 6
C 9
D 18
【解析】本题考查复合赋值运算符。a*=3;等价于a=a*3;，当a=6时，6*3=18，所以a的值变为18。
（2023年6月）如果a为int类型的变量，且a的值为6，则执行a%=4;之后，a的值会是（ ）
A 1
B 2
C 3
D 4
【解析】本题考查复合赋值运算符。a%=4;等价于a=a%4;，6%4=2，所以a的值变为2。

（2023年6月）在下列代码的横线处填写（ ），使得输出是"20 10"。 #include using namespace std; int main(){ int a=10,b=20; a= ; b=a+b; a=b-a; cout<
A a+b
B b
C a-b
D b-a
【解析】本题考查变量交换。要输出"20 10"，最终a=20，b=10。分析代码：b=a+b; a=b-a;。如果a=-10，则b=-10+20=10，a=10-(-10)=20。所以a的初始值应该是a-b=10-20=-10。
