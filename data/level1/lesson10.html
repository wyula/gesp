课堂练习

小玉家的电费

【题目描述】

夏天到了，各家各户的用电量都增加了许多，相应的电费也交的更多了。小玉家今天收到

了一份电费通知单。小玉看到上面写，据闽价电 [2006]27规定：

月用电量在150千瓦时及以下部分按每千瓦时 0.4463元执行，

月用电量在151∼ 400千瓦时的部分按每千瓦时0.4663执行，

月用电量在401千瓦时及以上部分按每千瓦时0.5663元执行;

小玉想自己验证一下，电费通知单上应交电费的数目到底是否正确呢。请编写一个程序，

已知用电总计，根据电价规定，计算出应交的电费应该是多少。

要求：使用scanf()和printf()完成。

【输入格式】

输入一个小数，精确到小数点后2位，表示用电总计（单位以千瓦时计），不超过10000。

【输出格式】

输出一个小数，保留到小数点后1位（单位以元计）。

【输入样例】

267.50

【输出样例】

121.7

课堂练习

思路分析

① 输入一个小数，表示用电量

double a;

scanf("%lf",&a);

电量 单价 ② 判断电量属于哪个等级并计算费用，保存到变量m中

double m=0;

<=150 0.4463

if(a<=150)

151~400 0.4663

m=a*0.4463;

>=401 0.5663 else if(a>=151&&a<=400)

m=150*0.4463+(a-150)*0.4663;

else if(a>=401)

m=150*0.4463+250*0.4663+(a-400)*0.5663;

③ 输出结果，保留一位小数

printf("%.1lf",m);

课堂练习

写一写

#include <iostream>
using namespace std;
int main(){
double a;
scanf("%lf",&a);
double m=0;
if(a<=150) m=a*0.4463;
else if(a>=151&&a<=400) m=150*0.4463+(a-150)*0.4663;
else if(a>=401) m=150*0.4463+250*0.4663+(a-400)*0.5663;
printf("%.1lf",m);
return 0;
}
