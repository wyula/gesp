02 真题演练

让我们通过一些真题来巩固所学知识。

真题1：选择题

（2023年6月） 在C++语言中，关于数组的说法，错误的是（）


A. 数组的下标从0开始

B. 数组的大小必须在定义时确定

C. 数组的元素必须是同一类型

D. 数组名表示数组第一个元素的地址


答案：B


解析：在C++中，数组的大小可以在运行时确定（使用new动态分配），但在栈上定义的数组大小必须在编译时确定。

真题2：选择题

（2023年6月） 以下代码的输出结果是？

int arr[] = {1, 2, 3, 4, 5};
cout << sizeof(arr) / sizeof(arr[0]) << endl;

A. 4

B. 5

C. 20

D. 不确定


答案：B


解析：sizeof(arr)返回数组总字节数，sizeof(arr[0])返回单个元素字节数，相除得到元素个数。

真题3：判断题

（样题） 枚举算法一定能找到问题的最优解。


答案：错误


解析：枚举算法可以找到所有满足条件的解，但不一定是最优解，除非问题本身就是求所有解。

真题5：选择题

（2023年6月） 表达式(5 << 2) & (3 | 4)的结果是（）


A. 0

B. 4

C. 16

D. 20


答案：C


解析：5 << 2 = 20（10100），3 | 4 = 7（00111），20 & 7 = 16（10000）

编程题1：数组排序

题目描述：

输入一个整数数组，使用选择排序算法对其进行升序排序。


输入格式：

第一行输入整数n

第二行输入n个整数


输出格式：

排序后的数组

数组排序参考代码

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int arr[100];
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    for (int i = 0; i < n - 1; i++) {
        int min_index = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_index]) {
                min_index = j;
            }
        }
        int temp = arr[i];
        arr[i] = arr[min_index];
        arr[min_index] = temp;
    }
    
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}

编程题2：字符串反转

题目描述：

输入一个字符串，将其反转后输出。


输入格式：

一个字符串


输出格式：

反转后的字符串

字符串反转参考代码

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;
    
    int left = 0, right = s.size() - 1;
    while (left < right) {
        char temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        left++;
        right--;
    }
    
    cout << s << endl;
    
    return 0;
}

编程题3：数字统计

题目描述：

输入一个整数，统计其二进制表示中0和1的个数。


输入格式：

一个整数n


输出格式：

0的个数和1的个数

数字统计参考代码

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    
    int count0 = 0, count1 = 0;
    while (n) {
        if (n & 1) {
            count1++;
        } else {
            count0++;
        }
        n = n >> 1;
    }
    
    cout << "0的个数：" << count0 << endl;
    cout << "1的个数：" << count1 << endl;
    
    return 0;
}
