第五课 枚举算法

01 什么是枚举算法

枚举算法也叫穷举算法，是一种基本的算法思想。它的核心思想是：


将问题的所有可能解一一列举出来，然后逐一验证每个解是否满足问题的条件。


枚举算法的特点：

1. 简单直观：容易理解和实现

2. 适用范围广：可以解决很多问题

3. 效率较低：时间复杂度通常较高

4. 结果可靠：只要枚举完全，一定能找到正确解

枚举算法的基本步骤

1. 确定枚举范围：确定所有可能的解的范围

2. 枚举每个可能的解：用循环遍历所有可能

3. 验证条件：检查当前解是否满足条件

4. 输出结果：如果满足条件，输出解

枚举算法流程图

开始

确定枚举范围 [start, end]

i = start

┌─ 循环: i <= end ─┐

│ 检查条件(i)是否满足 │

│ ├─ 是 → 输出/记录解 │

│ └─ 否 → 继续 │

│ i++ │

└─────────────────────┘

结束

02 枚举算法的应用场景

场景1：查找问题

- 在数组中查找某个元素

- 在字符串中查找子串


场景2：组合问题

- 排列组合问题

- 子集问题


场景3：优化问题

- 求最大值/最小值

- 求最优解


场景4：验证问题

- 判断是否存在满足条件的解

- 判断某个命题是否成立

示例1：查找最大值

#include <iostream>
using namespace std;

int main() {
    int arr[] = {5, 3, 9, 1, 7, 4};
    int len = sizeof(arr) / sizeof(arr[0]);
    
    int max_val = arr[0];
    for (int i = 1; i < len; i++) {
        if (arr[i] > max_val) {
            max_val = arr[i];
        }
    }
    
    cout << "最大值：" << max_val << endl;
    
    return 0;
}

示例2：百鸡问题

问题描述：

公鸡5元一只，母鸡3元一只，小鸡1元三只。用100元买100只鸡，问公鸡、母鸡、小鸡各买多少只？

百鸡问题代码

#include <iostream>
using namespace std;

int main() {
    for (int x = 0; x <= 20; x++) { // 公鸡数量
        for (int y = 0; y <= 33; y++) { // 母鸡数量
            int z = 100 - x - y; // 小鸡数量
            if (z % 3 == 0 && 5 * x + 3 * y + z / 3 == 100) {
                cout << "公鸡：" << x << "只，母鸡：" << y << "只，小鸡：" << z << "只" << endl;
            }
        }
    }
    
    return 0;
}
百鸡问题中，公鸡最多可以买多少只？

B 18只
C 20只
D 25只

03 枚举算法的优化

优化策略1：缩小枚举范围

根据问题的约束条件，尽可能缩小枚举的范围。


优化策略2：剪枝

在枚举过程中，提前排除不可能的情况，减少不必要的计算。


优化策略3：利用对称性

如果问题具有对称性，可以只枚举一半，另一半通过对称得到。

示例3：寻找水仙花数

问题描述：

水仙花数是指一个3位数，其各位数字的立方和等于该数本身。例如153 = 1³ + 5³ + 3³。

找出所有的水仙花数。

水仙花数代码

#include <iostream>
using namespace std;

int main() {
    cout << "水仙花数：" << endl;
    for (int num = 100; num < 1000; num++) {
        int hundreds = num / 100; // 百位
        int tens = (num / 10) % 10; // 十位
        int units = num % 10; // 个位
        
        if (hundreds * hundreds * hundreds +
            tens * tens * tens +
            units * units * units == num) {
            cout << num << " ";
        }
    }
    cout << endl;
    
    return 0;
}

04 真题解析

让我们来看一些关于枚举算法的真题。

判断题

（2023年6月） 枚举算法的时间复杂度一定是O(n)。


答案：错误


解析：枚举算法的时间复杂度取决于枚举的范围和每个解的验证时间，可能是O(n)、O(n²)等。

选择题

（样题） 枚举算法的主要优点是（）


A. 时间效率高

B. 空间效率高

C. 逻辑简单，容易实现

D. 不需要考虑边界条件


答案：C

编程题：完数

题目描述：

一个数如果恰好等于它的真因子之和，这个数就称为"完数"。例如6 = 1 + 2 + 3。

找出1000以内的所有完数。


输出格式：

每行输出一个完数

完数参考代码

#include <iostream>
using namespace std;

int main() {
    for (int num = 2; num < 1000; num++) {
        int sum = 1;
        for (int i = 2; i * i <= num; i++) {
            if (num % i == 0) {
                sum += i;
                if (i != num / i) {
                    sum += num / i;
                }
            }
        }
        if (sum == num) {
            cout << num << endl;
        }
    }
    
    return 0;
}

练习1： 找出1000以内的所有质数。


练习2： 输入两个整数，找出它们的最大公约数。


练习3： 找出所有满足a² + b² = c²的勾股数（a, b, c ≤ 100）。
