课堂练习 以下哪个函数是用来在 string 类中删除字符（ ）。
A empty()
B replace()
C substr()
D erase()
【解析】选项D正确。empty()用于判断字符串是否为空；replace()用于替换字符串；substr()用于截取子串；erase()用于删除字符。
课堂练习 执行以下代码，输出结果是( ) string s="Hello,C++"; string s1 = s.substr(3,5); cout << s1;
A llo,C+
B llo,C++
C Io,C+
D Io,C++
【解析】选项A正确。substr(pos, len)从位置pos开始截取长度为len的子串。s.substr(3,5)从索引3开始截取5个字符，即"llo,C"。
