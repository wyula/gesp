第九课 数制转换

01 数制概述

数制是计数的方式，不同的数制使用不同的基数。


常用数制：

1. 十进制（Decimal）：基数为10，使用0-9

2. 二进制（Binary）：基数为2，使用0和1

3. 八进制（Octal）：基数为8，使用0-7

4. 十六进制（Hexadecimal）：基数为16，使用0-9和A-F

不同数制的表示

1 1 1 1
10 1010 12 A
16 10000 20 10
255 11111111 377 FF

02 十进制转其他进制

转换方法：除基取余法

1. 将十进制数除以目标进制的基数

2. 记录余数

3. 将商继续除以基数

4. 重复步骤2-3，直到商为0

5. 将余数从后往前排列

十进制转二进制示例

#include <iostream>
#include <string>
using namespace std;

string decimalToBinary(int num) {
    if (num == 0) return "0";
    string result;
    while (num > 0) {
        result = to_string(num % 2) + result;
        num /= 2;
    }
    return result;
}

int main() {
    int num = 25;
    cout << num << "的二进制是：" << decimalToBinary(num) << endl;
    
    return 0;
}

十进制转十六进制示例

#include <iostream>
#include <string>
using namespace std;

char getHexChar(int n) {
    if (n < 10) return '0' + n;
    return 'A' + (n - 10);
}

string decimalToHex(int num) {
    if (num == 0) return "0";
    string result;
    while (num > 0) {
        result = getHexChar(num % 16) + result;
        num /= 16;
    }
    return result;
}

int main() {
    int num = 255;
    cout << num << "的十六进制是：" << decimalToHex(num) << endl;
    
    return 0;
}

03 其他进制转十进制

转换方法：按权展开法

1. 从右往左，每一位乘以基数的相应次幂

2. 将所有位的结果相加


示例：二进制1010转十进制

1010 = 1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 8 + 0 + 2 + 0 = 10

二进制转十进制示例

#include <iostream>
#include <string>
using namespace std;

int binaryToDecimal(string binary) {
    int result = 0;
    for (int i = 0; i < binary.size(); i++) {
        result = result * 2 + (binary[i] - '0');
    }
    return result;
}

int main() {
    string binary = "1010";
    cout << binary << "的十进制是：" << binaryToDecimal(binary) << endl;
    
    return 0;
}

04 二进制与八进制、十六进制的转换

二进制转八进制：

从右往左，每3位一组，不足3位左边补0

示例：101101 → 010 110 1 → 2 6 1 → 261(八进制)


二进制转十六进制：

从右往左，每4位一组，不足4位左边补0

示例：101101 → 0010 1101 → 2 D → 2D(十六进制)

二进制转十六进制示例

#include <iostream>
#include <string>
using namespace std;

char getHexChar(int n) {
    if (n < 10) return '0' + n;
    return 'A' + (n - 10);
}

string binaryToHex(string binary) {
    while (binary.size() % 4 != 0) {
        binary = "0" + binary;
    }
    string result;
    for (int i = 0; i < binary.size(); i += 4) {
        int val = 0;
        for (int j = 0; j < 4; j++) {
            val = val * 2 + (binary[i + j] - '0');
        }
        result += getHexChar(val);
    }
    return result;
}

int main() {
    string binary = "101101";
    cout << binary << "的十六进制是：" << binaryToHex(binary) << endl;
    
    return 0;
}

05 C++中的数制表示

C++中不同数制的字面量表示：

- 十进制：直接写数字，如 100

- 八进制：以0开头，如 0144（等于十进制100）

- 十六进制：以0x或0X开头，如 0x64（等于十进制100）

- 二进制（C++14及以上）：以0b或0B开头，如 0b1100100（等于十进制100）

C++数制表示示例

#include <iostream>
using namespace std;

int main() {
    int dec = 100; // 十进制
    int oct = 0144; // 八进制
    int hex = 0x64; // 十六进制
    int bin = 0b1100100; // 二进制(C++14)
    
    cout << "十进制：" << dec << endl;
    cout << "八进制：" << oct << endl;
    cout << "十六进制：" << hex << endl;
    cout << "二进制：" << bin << endl;
    
    cout << "八进制输出：" << oct << endl;
    cout << "十六进制输出：" << hex << hex << endl;
    
    return 0;
}

编程题：任意进制转换

题目描述：

输入一个十进制整数和目标进制，将其转换为目标进制的字符串表示。


输入格式：

两个整数，分别表示十进制数和目标进制（2-16）


输出格式：

转换后的字符串

任意进制转换参考代码

#include <iostream>
#include <string>
using namespace std;

char getChar(int n) {
    if (n < 10) return '0' + n;
    return 'A' + (n - 10);
}

string convert(int num, int base) {
    if (num == 0) return "0";
    string result;
    while (num > 0) {
        result = getChar(num % base) + result;
        num /= base;
    }
    return result;
}

int main() {
    int num, base;
    cin >> num >> base;
    cout << convert(num, base) << endl;
    
    return 0;
}
