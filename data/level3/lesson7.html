01 函数的参数传递

C++中有三种参数传递方式：


1. 值传递：

- 将实参的值复制一份给形参

- 函数内部对形参的修改不会影响实参


2. 引用传递：

- 将实参的引用传递给形参

- 函数内部对形参的修改会影响实参

- 使用&符号声明引用参数


3. 指针传递：

- 将实参的地址传递给形参

- 函数内部通过指针修改实参的值

- 使用*符号声明指针参数

值传递示例

#include <iostream>
using namespace std;

void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    cout << "交换前：x=" << x << ", y=" << y << endl;
    swap(x, y);
    cout << "交换后：x=" << x << ", y=" << y << endl;
    
    return 0;
}

引用传递示例

#include <iostream>
using namespace std;

void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    cout << "交换前：x=" << x << ", y=" << y << endl;
    swap(x, y);
    cout << "交换后：x=" << x << ", y=" << y << endl;
    
    return 0;
}

02 函数的返回值

函数的返回值类型：

- 可以是基本数据类型（int、float、char等）

- 可以是数组（但通常返回指针）

- 可以是字符串（返回指针或string）

- 可以是结构体

- 可以是void（无返回值）


注意事项：

- 不要返回局部变量的指针或引用

- 返回值类型必须与函数声明一致

返回指针示例

#include <iostream>
using namespace std;

int* createArray(int size) {
    int* arr = new int[size];
    for (int i = 0; i < size; i++) {
        arr[i] = i + 1;
    }
    return arr;
}

int main() {
    int* ptr = createArray(5);
    for (int i = 0; i < 5; i++) {
        cout << ptr[i] << " ";
    }
    cout << endl;
    delete[] ptr;
    
    return 0;
}

03 函数重载

函数重载是指在同一个作用域内，可以定义多个同名函数，但它们的参数列表必须不同（参数类型、参数个数或参数顺序不同）。


函数重载的条件：

1. 函数名相同

2. 参数列表不同（类型、个数、顺序）

3. 返回值类型可以相同也可以不同


函数重载的作用：

- 提高代码的可读性和可维护性

- 允许使用相同的函数名处理不同类型的数据

函数重载示例

#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int add(int a, int b, int c) {
    return a + b + c;
}

int main() {
    cout << "add(1, 2) = " << add(1, 2) << endl;
    cout << "add(1.5, 2.5) = " << add(1.5, 2.5) << endl;
    cout << "add(1, 2, 3) = " << add(1, 2, 3) << endl;
    
    return 0;
}
以下哪个函数可以与int add(int a, int b)构成重载？
A int add(int x, int y)
B double add(int a, int b)
C int add(int a, int b, int c)
D void add(int a, int b)

04 递归函数

递归函数是指在函数体内调用自身的函数。


递归函数的组成：

1. 递归条件：函数调用自身的条件

2. 终止条件：函数不再调用自身的条件


递归函数的特点：

- 代码简洁，逻辑清晰

- 可能导致栈溢出（递归深度太大）

- 时间复杂度较高

递归函数示例：阶乘

#include <iostream>
using namespace std;

int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

int main() {
    int n = 5;
    cout << n << "! = " << factorial(n) << endl;
    
    return 0;
}

递归函数示例：斐波那契数列

#include <iostream>
using namespace std;

int fibonacci(int n) {
    if (n == 1 || n == 2) {
        return 1;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

int main() {
    for (int i = 1; i <= 10; i++) {
        cout << fibonacci(i) << " ";
    }
    cout << endl;
    
    return 0;
}

05 真题解析

让我们来看一些关于函数的真题。

判断题

（2023年6月） 函数重载只要求函数名相同，参数列表可以相同。


答案：错误


解析：函数重载要求参数列表必须不同，只改变返回值类型不能构成重载。

选择题

（样题） 以下关于递归函数的说法，正确的是（）


A. 递归函数不需要终止条件

B. 递归函数一定会导致栈溢出

C. 递归函数必须有终止条件

D. 递归函数的时间复杂度一定比迭代低


答案：C

编程题：最大公约数

题目描述：

使用递归函数求两个正整数的最大公约数（GCD）。


输入格式：

两个正整数a和b


输出格式：

它们的最大公约数


提示：

使用欧几里得算法：gcd(a, b) = gcd(b, a % b)，当b=0时，a即为最大公约数。

最大公约数参考代码

#include <iostream>
using namespace std;

int gcd(int a, int b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}

int main() {
    int a, b;
    cin >> a >> b;
    cout << gcd(a, b) << endl;
    
    return 0;
}
