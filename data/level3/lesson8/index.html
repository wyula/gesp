（2023年9月）以下哪个不是C++语言中的运算符？（ ）
A ~
B ~~
C <
D <<
【解析】选项B正确。~是按位取反运算符，<是小于运算符，<<是左移运算符，~~不是C++运算符。

A 正确
B 错误

（2023年6月）一个 int 类型的值，做以下哪个操作，一定会变回原来的值？（ ）
A 左移3位，再右移3位。
B 右移3位，再左移3位。
C 按位或7，再按位与-8。
D 按位异或7，再按位异或7。
【解析】选项D正确。根据异或的性质：a^b^b=a，所以异或两次相同的值一定会恢复原值。
（2023年6月）如果 a 为 int 类型的变量，下列哪个表达式可以正确求出满足"小于等于 a 且是 4 的倍数"的整数中最大的？（ ）
A (a&(~3))
B ((a<<2)>>2)
C (a^3)
D ((a-1)|3)+1
【解析】选项A正确。~3将低2位置为0，a&(~3)将低2位清零，得到小于等于a且是4的倍数的最大整数。D是向上取整，B可能溢出。

运算符的优先级

课堂练习 以下程序运行后的输出结果是（ ）。
#include <iostream>
using namespace std;
int main(){
int x=3,y=2,z=1;
cout<<(x/y&~z);
return 0;
}
A 3
B 2
C 1
D 0
【解析】选项D正确。x/y=3/2=1，~z=~1=-2，1&-2=0。
课堂练习 以下程序运行后的输出结果是（ ）。
#include <iostream>
using namespace std;
int main(){
int a=4|3;
int b=5^7;
a<<=b;
cout<<a;
return 0;
}
A 7
B 16
C 28
D 12
【解析】选项C正确。a=4|3=7，b=5^7=2，a<<=2后a=7×4=28。
