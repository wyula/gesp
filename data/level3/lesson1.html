01 数组的定义

数组的概念

数组就是一组数据，是由一组具有相同数据类型的变量组成的。


数组中的变量称为数组的元素，元素的本质就是变量。


变量1 变量2 变量3 变量4 变量5
98 88 85 78 92
int类型

定义数组

定义数组需要确定：


1. 数组的名称：命名规则与变量相同

2. 数组元素的类型：所有元素必须是同一种类型

3. 数组的结构：比如数组的长度


语法格式：

数据类型 数组名[长度];

int a[5]; // 可以保存5个整型元素

数组长度的注意事项

数组长度的要求：


1. 只能是大于0的整数

2. 不能是变量和变量表达式


int n = 5;
int a[n]; // × 错误：不能使用变量
int a[n+2]; // × 错误：不能使用变量表达式
int a['3']; // ✓ 正确：'3'的ASCII码是51，是常量

02 数组的初始化

数组初始化

语法格式：

数据类型 数组名[长度] = {元素1, 元素2, ..., 元素n};

int a[5] = {98, 88, 85, 78, 92};

注意：

- 元素个数不能超过设置的长度数

- int a[3] = {98, 88, 85, 78, 92}; // × 错误：元素个数超过长度

数组其他初始化方式

方式一：省略数组长度

int a[] = {98, 88, 85, 78, 92};
// 元素：98 88 85 78 92
// 元素给定的情况，长度可以省略

方式二：自动补0

int b[5] = {1}; // 元素：1 0 0 0 0
int c[5] = {1, 2, 3}; // 元素：1 2 3 0 0

数组其他初始化方式（续）

方式三：数组元素全部初始化为0

int d[5] = {0}; // 元素：0 0 0 0 0
int e[5] = {}; // 元素：0 0 0 0 0
int f[5] = {0}, g[10] = {0}; // 同时初始化多个数组

方式四：同时定义多个数组

int h[5], n = 5;
// 数组h未初始化，变量n=5

03 数组的下标和元素访问

数组下标

数组元素在内存中是顺序、连续存储的，每个元素都有对应的编号，这个编号称为数组下标。


下标特点：

- 从0开始的一组整数

- 下标范围：0 ~ 数组长度-1


int a[5] = {98, 88, 85, 78, 92};
数组a: 98 88 85 78 92
下标: 0 1 2 3 4

输出数组中的元素

访问数组元素：

数组名称[下标表达式]

下标表达式的要求：

- 结果必须为整数


示例：

int a[5] = {98, 88, 85, 78, 92};

cout << a[0] << " "; // 输出98
cout << a[1] << " "; // 输出88
cout << a[2] << " "; // 输出85
cout << a[3] << " "; // 输出78
cout << a[4]; // 输出92

遍历数组元素

依次访问数组中每一个元素的操作，称为遍历数组元素。


使用for循环遍历：

int a[5] = {98, 88, 85, 78, 92};

for (int i = 0; i < 5; i++) {
    cout << a[i] << " ";
}
// 输出：98 88 85 78 92

数组a: 98 88 85 78 92

下标: 0 1 2 3 4

真题解析：小杨的储蓄（2023年9月）

问题描述

小杨共有N个储蓄罐，编号从0到N-1。从第1天开始，小杨每天都会往存钱罐里存钱。具体来说，第i天他会挑选一个存钱罐a_i，并存入i元钱。过了D天后，他已经忘记每个储蓄罐里都存了多少钱了，你能帮帮他吗？


输入描述：

输入2行，第一行两个整数N,D；第二行D个整数，其中第i个整数为a_i（保证0≤a_i≤N-1）。每行的各个整数之间用单个空格分隔。保证1≤N≤1000; 1≤D≤1000。


输出描述：

输出N个用单个空格隔开的整数，其中第i个整数表示编号为i-1的存钱罐中有多少钱（i=1,....N）。

样例输入输出

样例输入1：

2 3
0 1 0

样例输出1：

4 2

样例解释：

小杨在第1天、第2天、第3天分别向0号、1号、0号存钱罐存了1元钱、2元钱、3元钱，因此0号存钱罐有1+3=4元钱，而1号存钱罐有2元钱。


样例输入2：

3 5
0 0 0 2 0

样例输出2：

11 0 4

思路分析

解题思路：


1. 定义数组：创建一个数组来存储每个储蓄罐的金额

2. 输入数据：读取储蓄罐数量N和天数D

3. 模拟存钱过程：第i天向a_i号储蓄罐存入i元

4. 输出结果：依次输出每个储蓄罐的金额

步骤一：输入所需数据

#include <iostream>
using namespace std;

int jar[1000]; // 存钱罐数组

int main() {
    int n = 0, d = 0;
    cin >> n >> d;
    return 0;
}

步骤二：按题目规则进行存储统计

#include <iostream>
using namespace std;

int jar[1000];

int main() {
    int n = 0, d = 0;
    cin >> n >> d;
    
    int a = 0;
    for (int i = 1; i <= d; i++) {
        cin >> a;
        jar[a] += i; // 第i天向a号存钱罐存入i元
    }
    
    return 0;
}

步骤三：输出结果

#include <iostream>
using namespace std;

int jar[1000];

int main() {
    int n = 0, d = 0;
    cin >> n >> d;
    
    int a = 0;
    for (int i = 1; i <= d; i++) {
        cin >> a;
        jar[a] += i;
    }
    
    for (int i = 0; i < n; i++) {
        cout << jar[i] << " ";
    }
    
    return 0;
}

#include <iostream>

int jar[1000];

    }
    
    for (int i = 0; i < n; i++) {

    }
    
    return 0;
}
