（2023年6月）在C++语言中，定义数组时， [] 中必须指定元素个数。
A 正确
B 错误
【解析】数组定义语法：数据类型 数组名[长度]。长度必须是大于0的整数常量。可以省略长度，但必须提供初始值列表。
（样题）若有以下代码，则数组arr的长度是（ C ）。 int arr[] = {1, 2, 3, 4, 5};
A 3
B 4
C 5
D 6
【解析】数组定义时可以省略长度，编译器会根据初始值列表自动确定数组长度。这里有5个初始值，所以数组长度为5。
（2023年9月）以下数组定义，符合C++语言语法的是（ B ）。
A double a[];
B double b[] = {1,2.0,'3'};
C double c[3.0]
D double[] d = new double[3];
【解析】选项A：double a[]; 省略长度时必须提供初始值列表。选项B：double b[] = {1,2.0,'3'}; 正确，长度省略但提供了初始值。选项C：长度不能是浮点数。选项D：这是Java语法，C++不支持。

（2023年6月）在 C++语言中，长度为 n 的数组，合理的下标范围 从 0 到 n，包括 0 和 n。
A 正确
B 错误
【解析】数组下标从0开始计数。长度为n的数组，下标范围是0到n-1。不能使用浮点数作为下标。
（样题）在 C++语言中，数组的下标从 1 开始计数。
A 正确
B 错误
【解析】数组下标从0开始计数。长度为n的数组，下标范围是0到n-1。不能使用浮点数作为下标。
（样题）在 C++语言中，可以使用浮点数（如 3.0）作为数组下标。
A 正确
B 错误
【解析】数组下标从0开始计数。长度为n的数组，下标范围是0到n-1。不能使用浮点数作为下标。
（2023年6月）在 C++语言中，可以使用字符（如'0'）作为数组下标。
A 正确
B 错误
