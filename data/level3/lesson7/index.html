（2023年6月）以下哪个不是C++语言中的运算符？（ ）
A &
B &&
C *
D **
【解析】选项D正确。&是按位与运算符，&&是逻辑与运算符，*是乘法运算符或指针运算符。**不是C++中的运算符。
（2023年9月）如果a为int类型的变量，且表达式((a & 1) == 0) 的值为true，则说明a是偶数。
A 正确
B 错误
【解析】正确。二进制数的最低位为0表示偶数，为1表示奇数。a & 1可以获取最低位，结果为0说明是偶数。
（样题）C++语言中数字的符号位是不参与位运算的。
A 正确
B 错误
【解析】错误。位运算会对包括符号位在内的所有二进制位进行操作。负数以补码形式参与位运算。

按位或运算符|

或运算的用途

（2023年9月）如果 a是int类型的变量，下列哪个表达式的值一定为 true？（ ）
A a+1000-1000==a
B a*2/2==a
C (a&1)==1
D (a|1)==a+1
【解析】选项A正确。选项B中a*2可能导致整数溢出；选项C当a为偶数时结果为false；选项D当a为奇数时，a|1=a，不等于a+1。
（2023年6月）如果 a 为 int 类型的变量，且表达式((a | 3) == 3)的值为 true，则说明 a 在从 0 到 3 之间（可能为 0、可能为 3）。
A 正确
B 错误
【解析】错误。a | 3 == 3说明a的低2位都是0，但高位可以是任意值，例如a=4（100），4|3=7≠3；但a=8（1000），8|3=11≠3。实际上当a的低2位为0时，a可以是0、4、8、12等。

按位异或运算符^

异或运算符的性质和用途

（2023年6月）如果 a 和 b 均为 int 类型的变量，下列表达式能正确判断“a 等于 b”的是（ ）。
A ((a/b)==1)
B ((a&b)==a)
C ((a^b)==0)
D ((a|b)==b)
【解析】选项C正确。a^b==0说明a和b的每一位都相同，即a等于b。A中a/b==1不能判断相等（如5/3=1）；B中a&b==a只说明a是b的子集；D中a|b==b只说明b是a的超集。
（2023年6月）在下列代码的横线处填写（ ），可以使得输出是"24 12"。
#include <iostream>
using namespace std;
int main(){
int a=12,b=24;
____;//在此处填入代码
a=a^b;
b=a^b;
cout<<a<<" "<<b<<endl;
return 0;
}
A a=a^b
B b=a^b
C a=a+b
D b=a+b
【解析】选项B正确。这是经典的异或交换算法：第一步b=a^b=12^24；第二步a=a^b=12^(12^24)=24；第三步b=a^b=24^(12^24)=12。最终a=24,b=12。

#include <iostream>
using namespace std;
int main(){
int array[5]={1,2,4,8,16};
int res=0;
for(int i=0;i<5;i++)
____;//在此处填写代码

}
A res=res+array[i]
B res=res&array[i]
C res=res|array[i]

【解析】选项B正确。数组元素是1、2、4、8、16（二进制分别是00001、00010、00100、01000、10000）。A加法得31；C或运算得31；D异或运算得31；B与运算每次都是0&array[i]=0。

按位取反运算符~

运算符的优先级

（2023年9月）在C++语言中，位运算符也有类似“先乘除、后加减”的优先级规则。因此，使用时应注意合理使用括号。
A 正确
B 错误
【解析】正确。位运算符有明确的优先级：~（取反）优先级最高，然后是<<和>>，接着是&，然后是^，最后是|。合理使用括号可以避免优先级错误。
（样题）如果 a 和 b 均为 int 类型的变量，下列表达式能正确判断“a 等于 0 且 b 等于 0”的是（ ）。
A ((~a)&&(~b))
B ((a&b)==0)
C ((a|b)==0)
D ((a^b)==0)
【解析】选项C正确。a|b==0说明a和b的所有位都是0，即a=0且b=0。A中~a当a=0时为-1，非零为true，但无法判断b；B中a&b==0可能其中一个不为0；D中a^b==0只说明a==b，不一定都是0。
（样题）如果 a 为 int 类型的变量，下列哪个表达式可以正确求出满足“大于等于 a 且是 4 的倍数”的整数中最小的？（ ）
A (a&(~3))
B (a/4*4)
C ((a-1)|3)+1
D (a<<2)
【解析】选项C正确。A和B会向下取整（小于等于a的4的倍数）；D是乘以4；C的原理：(a-1)|3将低2位置为1，加1后进位到4的倍数，实现向上取整。
