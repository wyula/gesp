01 数组的常见操作

数组的常见操作包括：


1. 数组的复制：将一个数组的内容复制到另一个数组

2. 数组的合并：将两个数组合并成一个新数组

3. 数组的查找：在数组中查找特定元素

4. 数组的排序：对数组元素进行排序

数组复制

#include <iostream>
using namespace std;

int main() {
    int arr1[] = {1, 2, 3, 4, 5};
    int arr2[5];
    int len = sizeof(arr1) / sizeof(arr1[0]);
    
    for (int i = 0; i < len; i++) {
        arr2[i] = arr1[i];
    }
    
    cout << "arr2: ";
    for (int i = 0; i < len; i++) {
        cout << arr2[i] << " ";
    }
    cout << endl;
    
    return 0;
}

数组查找（顺序查找）

#include <iostream>
using namespace std;

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int len = sizeof(arr) / sizeof(arr[0]);
    int target = 30;
    int index = -1;
    
    for (int i = 0; i < len; i++) {
        if (arr[i] == target) {
            index = i;
            break;
        }
    }
    
    if (index != -1) {
        cout << "找到了，下标为：" << index << endl;
    } else {
        cout << "未找到" << endl;
    }
    
    return 0;
}

02 冒泡排序

冒泡排序是一种简单的排序算法，通过重复遍历数组，比较相邻元素并交换位置，使较大的元素逐渐'冒泡'到数组末端。


排序过程：

1. 比较第一个和第二个元素，如果第一个大于第二个则交换

2. 比较第二个和第三个元素，如果第二个大于第三个则交换

3. 继续这个过程，直到最后一个元素

4. 重复上述步骤，每次少比较一个元素

5. 直到没有需要交换的元素

冒泡排序实现

#include <iostream>
using namespace std;

int main() {
    int arr[] = {34, 7, 23, 32, 5, 62};
    int len = sizeof(arr) / sizeof(arr[0]);
    
    for (int i = 0; i < len - 1; i++) {
        for (int j = 0; j < len - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    
    cout << "排序后：";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}

03 选择排序

选择排序是一种简单直观的排序算法，每次从未排序部分选择最小（或最大）的元素，放到已排序部分的末尾。


排序过程：

1. 在未排序部分找到最小元素

2. 将最小元素与未排序部分的第一个元素交换

3. 缩小未排序部分的范围

4. 重复上述步骤，直到整个数组有序

选择排序实现

#include <iostream>
using namespace std;

int main() {
    int arr[] = {34, 7, 23, 32, 5, 62};
    int len = sizeof(arr) / sizeof(arr[0]);
    
    for (int i = 0; i < len - 1; i++) {
        int min_index = i;
        for (int j = i + 1; j < len; j++) {
            if (arr[j] < arr[min_index]) {
                min_index = j;
            }
        }
        int temp = arr[i];
        arr[i] = arr[min_index];
        arr[min_index] = temp;
    }
    
    cout << "排序后：";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    
    return 0;
}

04 真题解析

让我们来看一些关于数组排序的真题。

判断题

（2023年6月） 冒泡排序在最坏情况下需要进行n(n-1)/2次比较。


答案：正确


解析：冒泡排序最坏情况是数组完全逆序，需要比较(n-1)+(n-2)+...+1 = n(n-1)/2次。

选择题

（样题） 对数组{5, 3, 8, 1, 2}进行冒泡排序，第一轮排序后的结果是（）


A. {3, 5, 1, 2, 8}

B. {3, 5, 8, 1, 2}

C. {3, 5, 1, 8, 2}

D. {5, 3, 1, 2, 8}


答案：A

编程题：数组去重

题目描述：

输入一个有序数组，去除重复元素，输出去重后的数组。


输入格式：

第一行输入整数n

第二行输入n个整数（有序）


输出格式：

去重后的数组元素


样例输入：

6
1 2 2 3 3 3

样例输出：

1 2 3

数组去重参考代码

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int arr[100];
    
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    if (n == 0) return 0;
    
    cout << arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] != arr[i - 1]) {
            cout << " " << arr[i];
        }
    }
    cout << endl;
    
    return 0;
}
