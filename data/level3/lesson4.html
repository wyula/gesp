01 C++ string类

C++提供了string类，使用起来比字符数组更方便、更安全。需要包含头文件<string>。


string类的优点：

1. 不需要关心内存分配问题

2. 可以直接使用+进行字符串拼接

3. 可以直接使用==、!=、<、>等运算符比较

4. 提供了丰富的成员函数

string对象的定义和初始化

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1; // 空字符串
    string s2 = "Hello"; // 用字符串常量初始化
    string s3(s2); // 用另一个string对象初始化
    string s4(5, 'A'); // 5个'A'
    
    cout << "s1: \"" << s1 << "\"" << endl;
    cout << "s2: \"" << s2 << "\"" << endl;
    cout << "s3: \"" << s3 << "\"" << endl;
    cout << "s4: \"" << s4 << "\"" << endl;
    
    return 0;
}

string的常用操作

字符串拼接：

string s1 = "Hello";
string s2 = " World";
string s3 = s1 + s2; // "Hello World"
s1 += s2; // s1 = "Hello World"

字符串比较：

string s1 = "Apple";
string s2 = "Banana";
s1 == s2; // false
s1 < s2; // true（字典序）
s1 > s2; // false

字符串访问：

string s = "Hello";
s[0]; // 'H'
s.at(1); // 'e'
string s1 = "abc"; string s2 = "def"; s1 + s2的结果是（）
A "abc"
B "def"
C "abcdef"
D "defabc"

string的常用成员函数

长度相关：

- s.size() / s.length()：返回字符串长度

- s.empty()：判断字符串是否为空


查找相关：

- s.find(substr)：查找子串，返回起始位置

- s.rfind(substr)：从右向左查找子串

- s.find_first_of(chars)：查找第一个在chars中的字符


截取相关：

- s.substr(pos, len)：从pos开始截取len个字符

- s.erase(pos, len)：删除从pos开始的len个字符

- s.insert(pos, substr)：在pos位置插入子串

string成员函数示例

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "Hello, C++!";
    
    cout << "长度：" << s.size() << endl;
    
    int pos = s.find("C++");
    cout << "C++的位置：" << pos << endl;
    
    string sub = s.substr(0, 5);
    cout << "截取前5个字符：" << sub << endl;
    
    s.insert(5, " World");
    cout << "插入后：" << s << endl;
    
    s.erase(5, 7);
    cout << "删除后：" << s << endl;
    
    return 0;
}

02 字符串的大小写转换

C++提供了<cctype>头文件中的字符处理函数。


常用函数：

- toupper(c)：将小写字母转换为大写

- tolower(c)：将大写字母转换为小写

- isupper(c)：判断是否为大写字母

- islower(c)：判断是否为小写字母

- isalpha(c)：判断是否为字母

- isdigit(c)：判断是否为数字

大小写转换示例

#include <iostream>
#include <string>
#include <cctype>
using namespace std;

int main() {
    string s = "Hello, World!";
    
    for (int i = 0; i < s.size(); i++) {
        if (islower(s[i])) {
            s[i] = toupper(s[i]);
        }
    }
    cout << "大写：" << s << endl;
    
    for (int i = 0; i < s.size(); i++) {
        if (isupper(s[i])) {
            s[i] = tolower(s[i]);
        }
    }
    cout << "小写：" << s << endl;
    
    return 0;
}
下列关于string类的说法，正确的是（）
A string需要手动添加'\0'结尾
B string不能直接使用+拼接
C string的size()返回字节数
D string需要包含<cstring>头文件

03 字符串与数字的转换

数字转字符串：

int num = 123;
string s = to_string(num); // "123"

double d = 3.14;
string s2 = to_string(d); // "3.140000"

字符串转数字：

string s = "123";
int num = stoi(s); // 123

string s2 = "3.14";
double d = stod(s2); // 3.14

字符串与数字转换示例

#include <iostream>
#include <string>
using namespace std;

int main() {
    int num = 456;
    string s = to_string(num);
    cout << "数字转字符串：" << s << endl;
    
    string s2 = "789";
    int num2 = stoi(s2);
    cout << "字符串转数字：" << num2 << endl;
    
    string s3 = "3.14159";
    double pi = stod(s3);
    cout << "字符串转小数：" << pi << endl;
    
    return 0;
}

选择题

（2023年6月） 以下哪个不是C++ string类的成员函数？


A. size()

B. length()

C. strlen()

D. find()


答案：C


解析：strlen()是cstring头文件中的函数，不是string类的成员函数。

选择题

（2023年6月） 以下代码的输出结果是？

string s = "Hello";
cout << s.find('l') << endl;

A. 1

B. 2

C. 3

D. -1


答案：B


解析：find()返回首次出现的位置，'l'首次出现在下标2。

编程题：提取数字

题目描述：

从输入字符串中提取所有数字字符，并按顺序输出。


输入格式：

输入一个字符串


输出格式：

提取出的数字字符串


样例输入：

Hello123World456

样例输出：

123456

提取数字参考代码

#include <iostream>
#include <string>
#include <cctype>
using namespace std;

int main() {
    string s;
    cin >> s;
    
    string result;
    for (int i = 0; i < s.size(); i++) {
        if (isdigit(s[i])) {
            result += s[i];
        }
    }
    
    cout << result << endl;
    
    return 0;
}
