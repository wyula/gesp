（样题）已知数组 a 定义为 int a[100];，则赋值语句 a['0'] = 3; 会导致编译错误。
A 正确
B 错误
【解析】本题考查数组下标。字符'0'的ASCII值是48，所以a['0']=3等价于a[48]=3，这是合法的数组访问，不会导致编译错误。

（样题）如果有如下二维数组定义，则 a[0][3]的值为（ ） int a[2][2] = {{0, 1}, {2, 3}};
A 编译出错
B 1
C 3
D 0
【解析】本题考查二维数组的越界访问。数组a是2×2的，a[0][3]的列下标越界。但由于二维数组按行存储，a[0][3]等价于a[1][1]=3，所以值为3。
