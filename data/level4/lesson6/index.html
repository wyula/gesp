(2023年9月)在C++语言中，可以通过定义结构体，定义一个新的数据类型。
A 正确
B 错误
【解析】本题考查结构体的定义。结构体是用户自定义的数据类型，可以组合不同类型的数据。
课堂练习 设有以下说明语句，则下面的叙述不正确的是 （ ）。
A struct 是结构体类型的关键字 struct stu{
B struct stu 是用户定义的结构体类型 int a; float b;
C stutype 是用户定义的结构体类型 } stutype;
D a 和 b 都是结构体成员名
【解析】本题考查结构体定义。struct stu是结构体类型，stutype是结构体变量名，不是类型名。选项C错误。
课堂练习 当说明一个结构体变量时，系统分配给它的内存是（ ）
A 各成员所需内存量的总和，并遵守对齐补齐原则
B 结构中第一个成员所需内存量
C 成员中占内存量最大者所需的容量
D 结构中最后一个成员所需内存量
【解析】本题考查结构体内存分配。结构体变量的内存大小是各成员所需内存量的总和，并遵守对齐补齐原则。
课堂练习 结构体类型变量在程序执行期间（ ）
A 所有成员一直驻留在内存中
B 只有一个成员驻留在内存中
C 部分成员驻留在内存中
D 没有成员驻留在内存中
【解析】本题考查结构体变量的内存特性。结构体变量在程序执行期间，所有成员一直驻留在内存中。
(2023年9月)在C++语言中，可以定义结构体类型的数组变量，定义结构体时也可以包含数组成员。
A 正确
B 错误
【解析】本题考查结构体数组和结构体成员。在C++中，可以定义结构体类型的数组，结构体中也可以包含数组成员。
课堂练习 下面程序运行的结果是（ ） struct cmplx {
A 0 int x; int y; } cnum[2]={1,3,2,7};
B 1 printf("%d\n",cnum[0].y/cnum[0].x*cnum[1].x);
C 3
D 6
【解析】本题考查结构体数组。cnum[0]={1,3}，cnum[1]={2,7}。表达式cnum[0].y/cnum[0].x*cnum[1].x = 3/1*2 = 6。
课堂练习 根据下面的定义，能打印出字母 M 的语句是 （ ） struct person { char name[9]; int age;
A printf("%c\n",a[3].name); } person a[10] = { "John", 17,
B printf("%c\n",a[3].name[1]); "Paul", 19, "Mary", 18, "adam", 16 };
C printf("%c\n",a[2].name[1]);
D printf("%c\n",a[2].name[0]);
【解析】本题考查结构体数组访问。a[2].name是"Mary"，a[2].name[0]是'M'，所以选项D正确。
