（2023年9月）对包含 n 个元素的数组进行冒泡排序，平均时间复杂度一般为（ ）
A O(n)
B O(n log n)
C O(n2)
D 以上都不正确
【解析】本题考查冒泡排序的时间复杂度。冒泡排序的内层循环需要遍历未排序部分，平均情况下需要进行O(n2)次比较和交换操作，因此平均时间复杂度为O(n2)。
（2023年9月）对N个元素的数组执行插入排序算法，通常的时间复杂度为O(N2)。
A 正确
B 错误
【解析】本题考查插入排序的时间复杂度。插入排序在平均情况下需要进行O(n2)次比较和移动操作，因此平均时间复杂度为O(n2)。
（2023年12月）用递归法求的n阶乘，时间复杂度是 O(n)。
A 正确
B 错误
【解析】本题考查递归求阶乘的时间复杂度。递归求阶乘需要调用n次函数，每次调用执行常数时间操作，因此时间复杂度为O(n)。
课堂练习 在对n个元素进行冒泡排序的过程中，最好情况下的时间复杂度为（ ）
A O(1)
B O(log n)
C O(n2)
D O(n)
【解析】本题考查冒泡排序的最好时间复杂度。当数组已经有序时，冒泡排序只需进行一轮比较，没有交换发生即可提前结束，因此最好情况下的时间复杂度为O(n)。
课堂练习 插入排序平均时间复杂度为O(n2)，空间复杂度为O(1)。
A 正确
B 错误
【解析】本题考查插入排序的复杂度。插入排序平均时间复杂度为O(n2)，只需要常数级别的额外空间（用于临时变量），因此空间复杂度为O(1)。
课堂练习 在对n个元素进行直接插入排序的过程中，算法的空间复杂度为（ ）
A O(1)
B O(log n)
C O(n2)
D O(n)
【解析】本题考查插入排序的空间复杂度。插入排序只需要常数级别的额外空间（用于临时变量），不需要额外的数组或数据结构，因此空间复杂度为O(1)。
