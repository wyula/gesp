（样题）C++语言中的指针变量可以指向任何类型的数据。
A 正确
B 错误
【解析】本题考查指针类型匹配。指针变量的类型必须与所指向变量的类型一致，例如int*指针只能指向int类型变量，不能指向double类型变量。
（样题）以下哪个选项是 C++中正确的指针变量声明（ ）
A int *p;
B int p*;
C *int p;
D int* p*;
【解析】本题考查指针变量声明语法。正确的指针声明格式是"数据类型 *指针名;"，其中*表示该变量是指针类型。选项A符合语法规范。
（2023年9月）如果 n 为 int 类型的变量，一个指针变量定义为 int *p = &n; ，则下列说法正确的是（ ）。
A 指针变量p的值与变量n是相同的。
B 指针变量p的值与变量n的地址是相同的。
C 指针变量p指向的值为 'n' 。
D 指针变量p指向的值与变量n的地址是相同的。
【解析】本题考查指针变量的基本概念。int *p = &n; 表示p存储的是n的地址，即p的值等于&n。p指向的值是*p，等于n的值。所以选项B正确。

课堂练习 关于指针变量的创建和赋值正确的是哪一项（ ）
A int a=10;
B int a=10; int *p1=&a;
C double b=10.5; int *p1=&b;
D double b=10.5; int *p1; p1=&b;
【解析】本题考查指针变量的创建和赋值。选项B正确，先定义int变量a，然后定义int指针p1并初始化为a的地址。选项C和D错误，因为int指针不能指向double类型变量。
课堂练习 关于代码int a, *p = &a；下列语句描述正确的是（ ）。
A p 是指向一维数组的指针
B p 是指向 int 类型的指针
C p 存储的是 a 的数值
D p 是指向函数的指针，该函数返回 int 类型的数据
【解析】本题考查指针类型判断。int *p = &a; 定义了一个指向int类型的指针p，p存储的是变量a的地址，而不是a的数值。所以选项B正确。
课堂练习 阅读以下程序，说法错误的是（ ）。int a=50; int *p=&a;
A 创建p时，*表示p是一个指针
B p只可以存储int变量的地址
C p经过初始化，获得了变量a的地址
D p里存储的值为50
【解析】本题考查指针变量的存储内容。p是指针变量，存储的是变量a的地址，而不是a的值50。*p才是指向的值，等于50。所以选项D的说法错误。
（2023年9月）在C++语言中，指针变量在逻辑上指向另一个变量在内存中的位置，指针变量本身不占用内存。
A 正确
B 错误
【解析】本题考查指针变量的存储空间。指针变量本身也占用内存空间，在32位系统中占4字节，在64位系统中占8字节。所以题目说法错误。
（样题）在 C++中，指针变量的大小（单位：字节）是（ ）
A 2
B 4
C 8
D 与编译器有关
【解析】本题考查指针变量的大小。指针变量的大小与编译器有关：32位编译器下占4字节，64位编译器下占8字节。所以选项D正确。
（2023年6月）一个变量定义为int *p = nullptr;，则下列说法正确的是（ ）
A 该指针变量的类型为 int。
B 该指针变量指向的类型为 int。
C 该指针变量指向的内存地址是随机的。
D 访问该指针变量指向的内存会出现编译错误。
【解析】本题考查空指针概念。int *p = nullptr; 定义了一个指向int类型的空指针，p本身是int*类型，指向的类型是int。nullptr是C++关键字，表示空地址，不是随机地址。访问空指针会导致运行时错误而非编译错误。所以选项B正确。
（2023年6月）下列关于 C++语言中指针的叙述，不正确的是（ ）。
A 指针变量中存储的是内存地址。
B 定义指针变量时必须指定其指向的类型。
C 指针变量只能指向基本类型变量，不能指向指针变量。
D 指针变量指向的内存地址不一定能够合法访问。
【解析】本题考查指针的基本概念。指针变量可以指向指针变量，即可以定义指针的指针（int **pp）。所以选项C的叙述不正确。
（2023年12月）如果变量x的地址是0x6ffe14,下面C++代码执行以后输出的是（ ）。int *p=NULL; int x=2; p=&x; p++; cout<
A 0x6ffe11
B 0x6ffe14
C 0x6ffe18
D 0x6ffe15
【解析】本题考查指针运算。p=&x使p指向x的地址0x6ffe14。p++使指针移动一个int存储单元，在32位系统中int占4字节，所以p变为0x6ffe18。
课堂练习 阅读以下程序，下面选项中全部表示地址的是（ ）。int *p=NULL; int a=10; p=&a;
A a，p，*p
B *a，&a，*p
C p，*p，&a
D &a，p，&*p
【解析】本题考查地址和值的区别。&a是变量a的地址，p是指针变量存储的地址，&*p等价于p（先解引用再取地址）。而a是变量的值，*p是指针指向的值。所以选项D全部表示地址。
课堂练习 现有代码int k=5；int *pk=&k，下面不能正确执行的赋值语句是（ ）。
A *pk=10
B k=*pk+*pk
C k=*pk*2
D k=pk
【解析】本题考查指针赋值操作。pk是指针变量存储的是地址（通常是4或8字节），而k是int类型（4字节），将地址赋值给int变量会导致类型不匹配，不能正确执行。选项D错误。
课堂练习 阅读以下程序，想要输出结果为200，则横线处需要填写的代码是（ ）。int a = 1000; int *p = &a; ________________ cout << a << endl;
A *p = 200;
B p = 200;
C &a = 200;
D 200 = *p;
【解析】本题考查通过指针修改变量值。*p = 200; 通过指针p修改其所指向的变量a的值为200，这样输出a就是200。选项B将200赋值给指针本身（地址），选项C试图修改变量地址（非法），选项D将值赋给表达式（非法）。所以选项A正确。
