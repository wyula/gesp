（2023年3月）IPv4 的地址通常用"点分十进制" 的表示形式，形如（a.b.c.d），其中 a、b、c、d 都是 1~255 之间的十进制整数。
A 正确
B 错误
【解析】IPv4地址的每个十进制数范围是0~255，而不是1~255。0也是合法的IP地址组成部分，例如192.168.0.1中的0。
课堂练习 按照TCP/IP协议，接入互联网的每一台主机都有一个唯一的地址标识，这个地址标识是( )。
A 主机地址
B 网络地址
C IP地址
D 端口地址
【解析】IP地址是接入互联网的每台主机的唯一标识，用于在网络中定位和通信。主机地址和网络地址是IP地址的组成部分，端口地址用于区分同一主机上的不同服务。
课堂练习 IPv4地址由( )个二进制位组成。
A 16
B 32
C 48
D 8
【解析】IPv4地址由32位二进制数组成，通常采用点分十进制表示法，分为4个8位二进制数。IPv6地址由128位二进制数组成。
课堂练习 下列几个32位IP地址中，书写错误的是( )。
A 162.105.135.27
B 192.168.0.1
C 256.256.129.1
D 10.0.0.1
【解析】IP地址的每个十进制数范围是0~255。选项C中的256超出了这个范围，因此是错误的IP地址。
课堂练习 以下关于IP地址的说法正确的是( )。
A IP地址由两部分组成：网络号、主机号。
B A类IP地址的网络号有16位。
C IP地址通常表示为点分二进制形式。
D IP地址可以固化在硬件中，是独一无二的。
【解析】IP地址由网络号和主机号两部分组成。A类IP地址的网络号是8位，B类是16位。IP地址通常表示为点分十进制形式。IP地址是逻辑地址，可以配置和更改，不是固化在硬件中的。
课堂练习 IPv6占多少个二进制位？( )
A 32
B 64
C 128
D 256
【解析】IPv6地址由128位二进制数组成，采用冒号十六进制表示法，相比IPv4的32位地址提供了更多的地址空间。
（2023年6月）域名是由一串用点分隔的名字来标识互联网上一个计算机或计算机组的名称，CCF编程能力等级认证官方网站的域名是 gesp.ccf.org.cn，其中顶级域名是 gesp。
A 正确
B 错误
【解析】顶级域名是域名中最右边的部分。在gesp.ccf.org.cn中，顶级域名是.cn（中国国家域名），org是二级域名，ccf是三级域名，gesp是四级域名。
课堂练习 中国的国家顶级域名是( )。
A .cn
B .ch
C .chn
D .China
【解析】中国的国家顶级域名是.cn（Country Code Top-Level Domain），这是国际标准的中国国别域名。
