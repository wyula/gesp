（样题）在 C++语言中，字符串是以'\0'结尾的字符数组。
A 正确
B 错误
【解析】在C++中，传统的C风格字符串是以'\0'（空字符）结尾的字符数组。'\0'的ASCII值为0，用于标识字符串的结束位置。
课堂练习 字符数组中的最后一个字符必须是 '\0' 。
A 正确
B 错误
【解析】错误。字符数组不一定必须以'\0'结尾。只有当字符数组用于存储字符串时，才需要以'\0'结尾来标识字符串结束。如果只是普通的字符数组，不需要'\0'。

课堂练习 在C++中，以（ ）作为字符串结束标志。
A '\n'
B ' '
C '0'
D '\0'
【解析】选项D正确。C++中字符串是以'\0'（空字符，ASCII码为0）作为结束标志的。'\n'是换行符，' '是空格，'0'是字符0，都不是字符串结束标志。

完整代码 - check函数

#include <iostream>
using namespace std;
char line[101];
char pwd[101];
bool chk[10];

bool check(char str[], int l) {
    if (l < 6 || l > 12) return false;
    bool hasC = false, hasL = false, hasD = false, hasS = false;
    for (int i = 0; str[i] != '\0'; i++) {
        if ('A' <= str[i] && str[i] <= 'Z') {
            hasC = true;
        } else if ('a' <= str[i] && str[i] <= 'z') {
            hasL = true;
        } else if ('0' <= str[i] && str[i] <= '9') {
            hasD = true;
        } else if (str[i] == '!' || str[i] == '@' || str[i] == '#' || str[i] == '$') {
            hasS = true;
        } else return false;
    }
    if (!hasS) return false;
    if (hasC + hasL + hasD < 2) return false;
    return true;
}

完整代码 - main函数

int main() {
    int n = 0;
    while (cin.getline(line, 101)) {
        int len = 0;
        for (int i = 0; line[i] != '\0'; i++) {
            if (line[i] != ' ') {
                pwd[len++] = line[i];
            }
        }
        pwd[len] = '\0';
        chk[n++] = check(pwd, len);
    }
    for (int i = 0; i < n; i++) {
        cout << (chk[i] ? "YES" : "NO") << endl;
    }
    return 0;
}
