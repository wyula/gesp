课堂练习 微型计算机的问世是由于（ ）的出现。
A 中小规模集成电路
B 晶体管电路
C 超大规模集成电路
D 电子管电路
【解析】微型计算机的问世是由于超大规模集成电路的出现，第四代计算机采用超大规模集成电路技术。
（2023年6月）诞生于1958年的103机是中国第一台通用数字电子计算机，比1946年在美国诞生的第一台通用电子计算机ENIAC晚了十多年。
A 正确
B 错误
【解析】世界上第一台电子数字计算机ENIAC诞生于1946年。中国第一台通用数字电子计算机103机诞生于1958年，确实比ENIAC晚了十多年。
（2023年9月）神威·太湖之光超级计算机是中国自主研制的超级计算机，在全球超级计算机TOP500排行榜中多次荣膺榜首。
A 正确
B 错误
【解析】'神威·太湖之光'是中国自主研制的超级计算机，使用国产处理器，多次在全球超级计算机TOP500排行榜中位列第一。
（样题）计算机领域的图灵奖为了纪念（ ）科学家图灵。
A 英国
B 德国
C 瑞典
D 法国
【解析】艾伦·图灵是英国数学家、逻辑学家，被称为计算机科学之父、人工智能之父。
课堂练习 被誉为“计算机之父”的科学家是（ ）
A 图灵
B 巴贝奇
C 比尔·盖茨
D 冯·诺依曼
【解析】冯·诺依曼被称为"计算机之父"，提出了冯·诺依曼结构，即计算机系统由运算器、控制器、存储器、输入设备、输出设备五部分组成。
课堂练习 在下列关于图灵奖的说法中，不正确的是（ ）。
A 图灵奖是美国计算机协会于1966年设立的，专门奖励那些对计算机事业作出重要贡献的个人
B 图灵奖有“计算机界诺贝尔奖”之称
C 迄今为止，还没有华裔计算机科学家获此殊荣
D 图灵奖的名称取自计算机科学的先驱、英国科学家阿兰·图灵
【解析】姚期智是华裔计算机科学家，2000年图灵奖获得者，所以C选项错误。
（2023年9月）早期计算机内存不够大，可以将字库固化在一个包含只读存储器的扩展卡中插入计算机主板帮助处理汉字。
A 正确
B 错误
【解析】早期计算机内存较小，确实可以通过扩展卡（如汉卡）将字库固化在只读存储器中，帮助处理汉字显示。
课堂练习 以下不是存储设备的是（ ）
A 光盘
B 磁盘
C 固态硬盘
D 鼠标
【解析】鼠标是输入设备，用于向计算机输入操作指令，不是存储设备。光盘、磁盘、固态硬盘都是存储设备。
（2023年9月）近年来，线上授课变得普遍，很多有助于改善教学效果的设备也逐渐流行，其中包括比较常用的手写板，那么它属于哪类设备（ ）。
A 输入
B 输出
C 控制
D 记录
【解析】手写板是输入设备，用户通过手写板向计算机输入手写内容。
（2023年3月）以下不属于计算机输入设备的有（ ）。
A 键盘
B 音箱
C 鼠标
D 传感器
【解析】音箱是输出设备，用于播放声音。键盘、鼠标、传感器都是输入设备。
（2023年6月）以下不属于计算机输出设备的有（ ）。
A 麦克风
B 音箱
C 打印机
D 显示器
【解析】麦克风是输入设备，用于采集声音。音箱、打印机、显示器都是输出设备。
（2023年6月）计算机硬件主要包括运算器、控制器、存储器、输入设备和输出设备。
A 正确
B 错误
【解析】这是冯·诺依曼结构的五大组成部分，正确描述了计算机硬件系统。
（2023年9月）我们通常说的"内存"属于计算机中的（ ）。
A 输出设备
B 输入设备
C 存储设备
D 打印设备
【解析】内存（RAM）是计算机的存储设备，用于临时存储当前正在执行的程序和数据。
课堂练习 计算机主机是由CPU与（ ）构成的
A 控制器
B 运算器
C 输入、输出设备
D 内存储器
【解析】计算机主机由CPU（中央处理器）和内存储器组成。控制器和运算器是CPU的组成部分，输入输出设备属于外设。
课堂练习 计算机如果缺少（ ），将无法正常启动。
A 内存
B 鼠标
C U盘
D 摄像头
【解析】内存是计算机运行的基础，操作系统和程序都需要加载到内存中才能运行。缺少内存，计算机无法启动。
（2023年3月）计算机系统中存储的基本单位用B来表示，它代表的是（ ）。
A Byte
B Block
C Bulk
D Bit
【解析】B代表Byte（字节），是计算机存储的基本单位，1Byte=8Bit（位）。
课堂练习 1MB等于（ ）。
A 1000字节
B 1024字节
C 1000×1000字节
D 1024×1024字节
【解析】在计算机中，1MB=1024KB，1KB=1024B，所以1MB=1024×1024字节。
