（2023年9月）人们所使用的手机上安装的App通常指的是（ ）。
A 一款操作系统
B 一款应用软件
C 一种通话设备
D 以上都不对
【解析】App是Application的缩写，指的是应用软件，是为满足用户特定需求而开发的软件。
（2023年3月）在Windows系统中通过键盘完成对选定文本移动的按键组合是先Ctrl+X，移动到目标位置后按Ctrl+V。
A 正确
B 错误
【解析】Ctrl+X是剪切操作，将选中的文本移到剪贴板；Ctrl+V是粘贴操作，将剪贴板内容粘贴到目标位置。
（2023年3月）程序员用C、C++、Python、Scratch等编写的程序能在CPU上直接执行。
A 正确
B 错误
【解析】高级语言编写的程序需要经过编译或解释才能在CPU上执行，不能直接执行。
（2023年9月）C++是一种高级程序设计语言。
A 正确
B 错误
【解析】C++是一种面向对象的高级程序设计语言，相对于机器语言和汇编语言，它更接近人类自然语言。
（2023年6月）高级语言编写的程序需要经过以下（ ）操作，可以生成在计算机上运行的可执行代码
A 编辑
B 保存
C 调试
D 编译
【解析】编译是将高级语言源代码转换为机器语言可执行代码的过程，这是生成可执行程序的关键步骤。
（2023年6月）ChatGPT是OpenAI研发的聊天机器人程序，它能通过理解和学习人类的语言来进行对话，还能根据聊天的上下文进行互动，完成很多工作。请你猜猜看，下面任务中，ChatGPT不能完成的是（ ）。
A 改邮件
B 编剧本
C 擦地板
D 写代码
【解析】ChatGPT是语言模型，只能处理文本信息，无法进行物理操作如擦地板。改邮件、编剧本、写代码都是文本处理任务，可以完成。
